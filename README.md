This project is a workout tracking software built with a client-server architecture. It features a modern desktop application and a separate RESTful API backend to save everything persistently in a cloud database.

I chose to build the client using the Avalonia UI framework to ensure the application is fully cross-platform post-development.


🚀 How to run the project:

To get everything working, you will need to run both the server and the client side-by-side in two separate terminals.

1. Start the Backend API:

    Navigate to the API directory: cd Project/DB/LiftApi

    Run the server: dotnet run

    (Wait for the "Now listening on..." message to ensure the API is live).

2. Start the Client Application:

    Open a new terminal.

    Navigate to the application directory: cd Project/LiftProject

    Run the app: dotnet run

Everything should now be set up correctly to use Lift. Enjoy!
